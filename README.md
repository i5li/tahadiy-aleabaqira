# Tahadiy Aleabaqira

Interactive quiz game with multiple difficulty levels and scoring system.

> Built in 45 minutes as part of a team challenge to develop a complete game within 60 minutes.

## Features

- 4 question categories:
  - Easy: General Questions
  - Medium: Cultural Information
  - Hard: Intelligence Challenge
  - Mixed: Random Questions
- Settings: Music toggle, Sound effects toggle, Hint system
- Win condition: Reach 10 points
- Installable as PWA
- Exit confirmation to prevent progress loss
- Play Again and Main Menu navigation

## How to Play

1. Click Start Game
2. Select difficulty level
3. Answer questions correctly
4. First to 10 points wins

## Installation

Open [Tahadiy Aleabaqira](https://i5li.github.io/tahadiy-aleabaqira/) in a browser or deploy to a web server.

## Technologies

- HTML5
- CSS3
- JavaScript
